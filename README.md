# Basit Hesap Makinesi

Basit ve kullanımı kolay bir web tabanlı hesap makinesi uygulaması.

## Özellikler

- Temel matematiksel işlemler (toplama, çıkarma, çarpma, bölme)
- Klavye desteği
- Temiz ve modern arayüz
- C (Temizle) ve ← (Sil) fonksiyonları

## Kullanım

1. Dosyaları bir klasöre indirin
2. `index.html` dosyasını tarayıcıda açın
3. Hesap makinesini kullanmaya başlayın!

## Klavye Kısayolları

| Tuş | İşlem |
|-----|-------|
| 0-9 | Sayıları girin |
| +, -, *, / | Operatörleri seçin |
| Enter (=) | Hesaplamayı yapın |
| Escape (C) | Ekranı temizleyin |
| Backspace (←) | Son karakteri silin |

## Dosya Yapısı

```
basit-hesap-makinesi/
├── index.html    # Ana HTML dosyası
├── style.css     # Stil dosyası
├── script.js     # JavaScript dosyası
└── README.md     # Bu dosya
```
