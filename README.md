# Basit Hesap Makinesi

Basit ve kullanımı kolay bir web tabanlı hesap makinesi uygulaması.

## Özellikler

- Temel matematik işlemleri: Toplama, çıkarma, çarpma, bölme
- Klavye desteği
- Modern ve kullanıcı dostu arayüz
- Hata kontrolü (sıfıra bölme, geçersiz işlemler)

## Kullanım

### Tarayıcıda Açma

1. `index.html` dosyasını bir web tarayıcısında açın
2. Hesap makinesini kullanmaya başlayın

### Klavye Kısayolları

| Tuş | İşlem |
|-----|-------|
| `0-9` | Sayı girişi |
| `.` | Ondalık nokta |
| `+ - * /` | İşlem operatörleri |
| `Enter` veya `=` | Hesapla |
| `Escape` veya `C` | Ekranı temizle |
| `Backspace` | Son karakteri sil |

## Dosya Yapısı

```
basit-hesap-makinesi/
├── index.html      # Ana HTML dosyası
├── style.css       # Stil dosyası
├── script.js       # JavaScript mantığı
└── README.md       # Bu dosya
```

## Geliştirme

Projeyi geliştirmek için:

1. Dosyaları bir metin editöründe açın
2. Değişiklikleri yapın
3. Tarayıcıda yenileyerek sonucu görün

## Teknolojiler

- HTML5
- CSS3
- Vanilla JavaScript (ES6+)
