# Basit Hesap Makinesi

Modern ve koyu tema ile tasarlanmış basit bir hesap makinesi uygulaması.

## Özellikler

- Toplama, çıkarma, çarpma ve bölme işlemleri
- Koyu tema, modern görünüm
- Klavye desteği
- Hata kontrolü

## Kullanım

Tarayıcıda `index.html` dosyasını açarak hesap makinesini kullanabilirsiniz.

### Klavye Kısayolları

- `0-9`: Rakam girişi
- `+ - * /`: İşlemler
- `Enter`: Hesapla
- `Escape`: Temizle
- `Backspace`: Son karakteri sil

## Dosya Yapısı

- `index.html`: Ana HTML dosyası
- `style.css`: Stil dosyası
- `script.js`: JavaScript kodu
